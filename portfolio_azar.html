SQL 기반 퍼널 바틀넥 진단 및 활성 유저 볼륨 극대화

Approach

데이터 분석의 기반에는 데이터 구조(ERD)에 대한 이해가 있습니다. 유저·결제 로그 등 여러 테이블이 어떻게 연결되는지를 먼저 파악하고, 그 구조 위에서 SQL을 도구로 활용해 분석을 설계합니다. 이 프로젝트는 그 역량이 실제 퍼널 병목 진단으로 이어진 사례입니다.

Problem

신규 획득 유저가 활성 유저로 전환되는 퍼널에서, 테이블 간 관계를 기반으로 결제 흐름을 분석한 결과 '첫 결제(First Purchase)'가 핵심 병목 지점임을 진단했습니다.

퍼널 병목 진단

Strategy

  • 여러 테이블(유저·결제 로그)의 관계를 이해한 상태에서 SQL 조인 쿼리로 고관여 미결제 유저 세그먼트를 직접 추출
  • 타겟팅 프로모션(CRM) 전개: 첫 결제 시 50% 캐시백 자동화
 high_intent_non_payers.sql
WITH user_activity AS (
  SELECT
    u.id AS user_id,
    COUNT(DISTINCT v.episode_id) AS viewed_episodes,
    COUNT(DISTINCT v.title_id) AS viewed_titles,
    SUM(CASE WHEN p.id IS NOT NULL THEN 1 ELSE 0 END) AS payment_cnt
  FROM users u
  LEFT JOIN view_log v ON u.id = v.user_id
    AND v.viewed_at >= NOW() - INTERVAL '30 days'
  LEFT JOIN payment_log p ON u.id = p.user_id
  GROUP BY u.id
)
SELECT user_id, viewed_episodes, viewed_titles
FROM user_activity
WHERE payment_cnt = 0  -- 미결제 유저
HAVING viewed_episodes >= 20  -- 고관여 임계치
ORDER BY viewed_episodes DESC;

users·view_log·payment_log 테이블의 관계를 이해한 뒤, CTE로 유저별 30일 열람량과 결제 여부를 집계하고 '활동량은 높지만 미결제'인 고관여 유저 세그먼트를 직접 도출한 쿼리입니다.

Result

고관여 미결제 유저 타겟팅으로 DAU/MAU가 113% 대폭 개선되었고, 타겟 프로모션 전개로 첫 결제 전환율(CVR)이 13%p 상승했습니다.

Approach

  • 친구 태그 캠페인 (Facebook / Instagram): 참여자가 댓글에 콘텐츠를 공유하고 친구를 태그해 함께 참여하도록 유도 — 참여자 각자의 친구 네트워크로 브랜드 노출 확장 (UGC + 초대 결합)

06 · Measurement Strategy
기여도 100%

ATT 이후 iOS 측정 환경 변화 대응 및 퍼스트파티 데이터 기반 성과 분석 체계 구축

Problem

Apple의 ATT(App Tracking Transparency) 도입 이후 iOS 캠페인의 사용자 단위 측정이 어려워지면서, 앱 캠페인 데이터만으로는 유저의 실제 가치를 정확히 평가하기 어려운 환경이 되었습니다.

Hypothesis

  • SKAN(SKAdNetwork) 기반의 제한적 집계 데이터만으로는 한계가 명확하다 — 퍼스트파티 User ID를 축으로 측정 체계를 재설계해야 ATT 환경에서도 유저의 장기 가치를 추적할 수 있다
측정 전략 전환 흐름

Action

  • iOS 측정 한계를 보완하기 위해 Web-to-App 유입 비중 확대
  • Singular 연동 데이터를 바탕으로 Snowflake 기반 User ID 분석 환경 구축

Analysis

  • 퍼스트파티 User ID를 축으로 가입 → 결제 → 재결제 경로를 연결해 유저 단위 장기 가치 추적
  • 국가별·플랫폼별 구매 행동을 세분화해 분석

Result

ATT 환경에서도 퍼스트파티 데이터 기반으로 유저의 장기 가치를 분석할 수 있는 체계를 확보했고, 단기 CPA 중심에서 재결제·ARPPU 중심 KPI 운영으로 전환할 근거를 마련했습니다.

Key Insight플랫폼이 부과한 측정 제약(ATT)은 서드파티 트래킹의 한계일 뿐, 퍼스트파티 User ID를 자산으로 삼으면 오히려 단기 지표를 넘어선 장기 가치 중심의 측정 체계로 전환하는 계기가 됩니다.

Strategy

장르 × 소재 포맷별 효율 비교

08 · Performance UA
기여도 100%

데이터 기반 퍼포먼스 UA 최적화 — 매체 다각화 · 콘텐츠 친화도 타겟팅 · 행동 데이터 리타겟팅

Problem

Meta·Google 양대 매체 의존(각 CPA +15%·+12%)과 단순 광범위 타겟팅으로는 신규 유저 획득 비용(CPA)과 전환 효율이 구조적으로 악화되는 상황이었습니다.

Strategy

  • 매체 다각화: TikTok·Pinterest를 신규 도입해 4채널 믹스(Meta 35 / TikTok 25 / Google 25 / Pinterest 15)로 재편
  • 콘텐츠 친화도 타겟팅: 유저 행동 데이터로 오디언스를 세분화(복귀작·유사 장르 기열람자)해 단순 광범위 타겟팅을 대체
  • 행동 데이터 리타겟팅: 플랫폼 외부의 오가닉 검색 인텐트(Anime 연관 키워드)를 포착해 IP 매칭 리타겟팅으로 연결
타겟팅 방식별 ROAS — 행동 데이터 기반의 압도적 효율

Result

  • 매체 다각화로 신규 유저 획득 CPA YoY 32% 절감, Meta/Google 의존 구조 탈피
  • 콘텐츠 친화도 타겟팅으로 ROAS 최대 229% 상승(유사 장르), 핵심 액션 장르 매출 TOP 3 달성
  • 행동 데이터 리타겟팅으로 연관 키워드 오가닉 검색량 10배, 소셜 방문자 ROAS MoM +132%
Key Insight콘텐츠 속성과 유저의 과거 소비·검색 인텐트 데이터를 결합한 '콘텐츠 친화도 타겟팅'이 단순 광범위 타겟팅보다 압도적인 ROAS를 만듭니다.
